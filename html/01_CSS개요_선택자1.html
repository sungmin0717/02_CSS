CSS(Cascading Style Sheets)

    마크업 언어(HTML)가 실제 화면에 표시되는 방법을 
    기술하는 언어

    하나의 요소에 여러 css 코드를 겹쳐서 원하는 스타일을 만들어냄.

    W3C라는 기관 에서 CSS 를 웹 표준으로 지정함.
CSS 선택자 (CSS Selector)

CSS 코드를 작성하는 방법

  1. head태그에 style 태그를 작성한 후 내부에 CSS 코드 작성
  2. CSS가 적용되길 원하는 요소에 직접 CSS 코드를 작성하는 방법 (inline-style)
  3. 외부에 .css 파일을 생성하여 link 태그를 이용해 연결하는 방법

id(#) 선택자

    HTML 문서 내에서 
    id 속성 값이 일치하는 요소를 선택하는 선택자

    작성법

    
      #id 속성값 { CSS 코드;}
  • id 선택자 테스트1
  • id 선택자 테스트2
  • id 선택자 테스트3

태그(태그명) 선택자

    HTML 문서 내에서
    태그명이 일치하는 모든 요소를 찾아
    선택하는 선택자

    작성법
    
      태그명 { CSS 코드;}

class(.) 선택자

    HTML 문서 내에서
    class 속성 값이 일치하는 
    모든 요소를 선택하는 선택자

    *하나의 요소가 여러 class를 가질 수 있다

    [작성법]
    
    
      .class속성값 { CSS 코드값;}
    

    
      요소에 css 적용을 위해 사용하는 선택자
      보통 class 선택자를 가장 많이 이용한다!

      (id 속성은 JS(JavaScript) 용도로 많이  사용함)

모든 요소(*, asterisk) 선택자

    단독 사용          : 문서에 존재하는 모든 요소 선택
    
      * { CSS 코드;}
    

    다른 선택자와 혼용 : 선택된 요소의 모든 자식/후손/형제 선택 
    
      A 선택자 * {CSS 코드;}
